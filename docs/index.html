Four things it refuses to fake

Falsification-first, not validation-first.

Architectural invariant

Deterministic safety floors run before the LLM and can bypass it. No LLM output overrides a floor's outcome — ever.

Append-only provenance

The Aletheia chain is signed and hash-linked. Truncation paths, unsigned entries, or missing prev_hash linkage are rejected.

Frozen benchmark

VBX-ISPS-BENCH v1.1 is cryptographically committed for the v1.x major. If a candidate fails a floor, the candidate is fixed — not the benchmark.

Honest epistemic markers

Every load-bearing claim carries VERIFIED, PLAUSIBLE, or SPECULATIVE. Markers are never stripped to make a document read more confidently.
